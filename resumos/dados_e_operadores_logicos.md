# Tipos de dados e operadores lógicos
## O que é um tipo de dado?
-  A classificação/categoria de um dado.
### Number
- Trata de números.


------------


> verificando tipo do dado com console.log

```javascript
console.log(typeof 13)
```
## Operações aritméticas com Numbers

- soma (+)
- subtração(-)
- DIvisão (/)
- Multipicação (*)
- Resto(%)

> Operações aritméticas com Numbers

```javascript
console.log(2+2)
console.log(4-3)
console.log(15*2)
console.log(400/4)
console.log(20%2)
```
